Typora使用手册-Markdown语法

Typora 语法全面介绍

Typora 是一款所见即所得的 Markdown 编辑器，支持多种格式和功能。本文作为学习笔记将从基础语法、文本格式、列表、引用、代码与公式、表格、图片与链接、特殊元素等几个板块进行全面介绍。

(一）基础语法

1. 标题

使用 # 号表示不同级别的标题，最多支持六级标题。

2. 段落与换行

段落之间需要空行分隔，换行使用两个空格加回车。

3. 分割线

使用三个或以上的 -、* 或 _

(二）文本格式

1. 强调

使用 * 或 _ 包围文字表示斜体，使用 ** 或 __ 表示加粗，使用 ~~ 表示删除线。

2. 脚注

使用 [^注释] 插入脚注，并在文末添加脚注内容。

3. 表情符号

Typora 支持标准的 Emoji 表情，可以直接输入相应的字符。

(三）列表

1. 无序列表

使用 -、* 或 + 表示无序列表。

2. 有序列表

使用数字加点表示有序列表。

3. 任务列表

使用 - [ ] 创建未完成项，使用 - [x] 创建已完成项。

(四）引用与代码

1. 引用

使用 > 表示引用，可以嵌套。

2. 行内代码

使用反引号包围行内代码。

3. 代码块

使用三个反引号包围代码块，并可指定语言。

(五）数学公式

使用美元符号 $ 包围行内公式，使用两个美元符号 $$ 包围块级公式。

$E = mc^2$

$$
E = mc^2
$$

1.上标和下标

在代码块中如果使用上标则在上标前加上^(内容)，即将内容作为上标，同理下标的符号是’_’。如果不在代码块内使用上标或下标则利用HTML语法，Typora支持HTML语法

上标 (内容)

下标 (内容)

(六）表格

使用 | 分隔表格内容，使用 - 创建表头。

(七）图片与链接

1. 链接

使用 [显示文本](链接地址) 表示超链接。

2. 图片

使用 ![替代文本](图片地址) 插入图片。

(八）特殊元素

1. 注释

使用 <!-- 注释内容 --> 插入注释，这些内容不会在最终文档中显示。

2. 表情符号

使用 :表情名称: 表示表情符号。
